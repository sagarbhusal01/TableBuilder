# Table builder for java cli 
  This package can be used to create tables in the command line .
  Table builder allows you to create tables like 


          ================================================
          ┃     ID         ┃     Data                    ┃
          ================================================
          ┃     four       ┃     what's up ladies!       ┃
          ┃     One        ┃     hello boys!!            ┃
          ┃     Three      ┃     hello Girls             ┃
          ┃     Two        ┃     hello Girls kbv         ┃
          ┃     five       ┃     hello again!            ┃
          ▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔

or like 

          =====================================================
          ┃          ID         ┃     Data                    ┃
          =====================================================
          ┃ 1. ┃     four       ┃     what's up ladies!       ┃
          ┃ 2. ┃     One        ┃     hello boys!!            ┃
          ┃ 3. ┃     Three      ┃     hello Girls             ┃
          ┃ 4. ┃     Two        ┃     hello Girls kbv         ┃
